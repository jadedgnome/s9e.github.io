OptimizeNestedConditionals

class OptimizeNestedConditionals extends AbstractNormalization

Optimize xsl:choose elements by integrating the content of another xsl:choose element located in their xsl:otherwise part

Will move child nodes from //xsl:choose/xsl:otherwise/xsl:choose to their great-grandparent as long as the inner xsl:choose has no siblings. Good for XSLT stylesheets because it reduces the number of nodes, not-so-good for the PHP renderer when it prevents from optimizing branch tables by mixing the branch keys
